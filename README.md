# Fork off - Will Smith (Chrome extension)

Overlays Will Smith on AI-generated images hosted on Forkable so you know what's up.

<img src="./screenshot.png" width="450" />

## Installation

1. Download or clone this folder to your machine.
2. Open Chrome and navigate to `chrome://extensions`.
3. Toggle **Developer mode** (upper-right corner).
4. Click **Load unpacked** and select the `forkable-chrome-ext` directory.
5. Visit `forkable.com` (or a subdomain). AI-generated images will have an overlay on top of them.

## Notes

- The extension runs only on the `forkable.com` domain
